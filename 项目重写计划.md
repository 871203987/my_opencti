# OpenCTI GraphQL 后端 Java 重写需求分析文档（详细版）

## 一、项目概述

### 1.1 项目背景
OpenCTI是一个开源的威胁情报平台，其核心后端服务 `opencti-graphql` 采用 Node.js + TypeScript 开发。现需要在 `opencti-java` 目录下使用 Java 21 进行一比一重写。

### 1.2 项目规模统计

| 指标 | 数量 |
|------|------|
| 源代码行数 | ~693,893 行 |
| TypeScript 文件 | 200+ 个 |
| JavaScript 文件 | 200+ 个 |
| 导出函数/类/接口 | 1,122+ 个 |
| 测试文件 | 100+ 个 |
| GraphQL Schema 定义 | 1个大型schema文件 |
| 数据库迁移脚本 | 60+ 个 |
| Python 脚本 | 7个 |

使用 **ProcessBuilder** 调用Python脚本，涉及位置需在代码中明确注释：
